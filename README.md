🗺️ AI Certification Roadmap 2026

A dynamic, interactive, and visual roadmap of formal, proctored Artificial Intelligence and Machine Learning certifications.

Inspired by the legendary Security Certification Roadmap by Paul Jerimy, this project aims to help professionals navigate the rapidly evolving landscape of AI certifications.


✨ Features

100% Client-Side: Built with pure HTML, CSS (Grid/Flexbox), and Vanilla JavaScript. No build tools or backend required.

Interactive Filters: Instantly filter certifications by Career Path (Strategy, Builder, Engineer, Trust/Risk) or Ecosystem/Provider (AWS, Microsoft, Google, Databricks, etc.).

Market Signals: Each certification is tagged with a market signal (e.g., "Top Market Signal", "Compliance Standard") to indicate its current industry value.

Smart Tooltips: Hover over any certification card to see its price and a quick description. The UI automatically adjusts tooltip positioning to prevent screen overflow.

🤝 Contributing

The AI landscape changes incredibly fast. If you notice an outdated certification, or want to propose a new one that has gained industry traction, your contributions are highly welcome!

Rules for Inclusion:

The certification must be formal and proctored (supervised).

We do not include video course completion diplomas or simple participation badges (e.g., standard Coursera/Udemy certificates without a proctored exam).

The certification must have an established market position.

How to propose a change:

Fork the project.

Edit the certData array inside the <script> tag in index.html.

Submit a Pull Request with a link to the official exam page.
